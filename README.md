# Design Playground

Внутренняя платформа Gigonom для проектирования, стандартизации и поставки дизайн-системы в продуктовые проекты.

Проект создается как чистый монорепозиторий на `pnpm`, `Turborepo` и `Next.js`. Цель — собрать сырьевые компоненты из существующих проектов, привести их к единому дизайн-контракту через Figma и выпускать переиспользуемые пакеты для Next.js/React-проектов команды.

## Целевой стек

- Node.js `>=24`
- pnpm `10.24.0`
- Turborepo
- Next.js `16`
- React `19`
- Tailwind CSS `4`
- Biome
- Steiger/FSD
- Storybook `10`
- Playwright для визуальных и интерактивных проверок

## Текущий статус

Текущий этап: `Этап 0: чистый фундамент`.

До завершения нулевого этапа не переносим компоненты и не проектируем публичный API пакетов. Сначала фиксируем архитектуру, источники, критерии готовности и правила инвентаризации.
