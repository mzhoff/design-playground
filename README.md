# Design Playground

Внутренняя платформа Gigonom для проектирования, стандартизации и поставки дизайн-системы в продуктовые проекты.

Проект создаётся как чистый `pnpm + Turborepo + Next.js` монорепозиторий. Цель — собрать сырьевые компоненты из существующих проектов, привести их к единому дизайн-контракту через Figma и выпускать переиспользуемые пакеты для Next/React-проектов команды.

## Целевой стек

- Node.js `>=24`
- pnpm `10.24.0`
- Turborepo
- Next.js `16`
- React `19`
- Tailwind CSS `4`
- Biome
- Steiger/FSD
- Storybook `10`
- Playwright для визуальных и интерактивных проверок

## Статус

Текущий этап: `Stage 0 — Foundation & RFC`.

До завершения Stage 0 не переносим компоненты и не проектируем публичный API пакетов. Сначала фиксируем архитектуру, источники, критерии готовности и правила инвентаризации.
