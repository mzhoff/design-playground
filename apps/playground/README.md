# apps/playground

Next.js-приложение для настройки визуального слоя дизайн-системы.

## Будущие задачи

- выбор пресета проекта;
- настройка базовых и семантических токенов;
- живой предпросмотр на реальных вертикальных наборах;
- экспорт CSS-переменных;
- экспорт фрагмента темы для Tailwind CSS 4;
- экспорт JSON-пресета;
- экспорт данных, совместимых с Token Studio или похожим процессом.

## Роль приложения

Playground — это продуктовый интерфейс конфигуратора.

В идеальном сценарии менеджер или дизайнер открывает Playground вместе с клиентом, настраивает стиль проекта, показывает его на реальных компонентах, согласует направление и экспортирует готовый пресет для разработки.

Playground не должен ломать структуру компонентов. Он меняет только те параметры, которые заранее разрешены дизайн-системой: цвета, типографику, радиусы, отступы, обводки, иконки, изображения и анимационные пресеты.
