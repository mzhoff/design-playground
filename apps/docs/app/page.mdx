# Design Playground

Design Playground — внутренняя платформа Gigonom для проектирования, настройки и поставки дизайн-системы в проекты заказной разработки.

## Что уже зафиксировано

- Next.js является основной runtime-целью.
- Компоненты и паттерны проектируются с учетом Feature-Sliced Design.
- Сырьевые компоненты сначала переносим вертикальными наборами.
- Финальная реализация делается после Figma-ревью.
- Playground отвечает за настройку стиля и экспорт пресетов.
- Storybook отвечает за техническую документацию и QA.
