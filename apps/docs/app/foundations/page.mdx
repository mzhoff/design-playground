# Foundations

Foundations — это базовые слои дизайн-системы, которые позже будут управляться токенами.

## Будущие слои

- цветовые шкалы;
- семантические цвета;
- типографика;
- отступы;
- радиусы;
- обводки;
- тени;
- анимационные пресеты;
- иконки и ассеты.

## Важное ограничение

На этапе runtime-каркаса мы не финализируем foundations. Сейчас только готовим место, куда они будут добавлены после инвентаризации и Figma-ревью.
