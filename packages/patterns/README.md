# packages/patterns

Пакет для вертикальных наборов интерфейса.

Именно с вертикальных наборов начинается инвентаризация, потому что они отражают реальные продуктовые задачи, а не абстрактные атомы.

Первые кандидаты:

- чат-ассистент;
- канвас;
- админские CRUD-интерфейсы;
- формы;
- WYSIWYG и текстовые сценарии;
- блоки сайта;
- анимационные паттерны.

После стабилизации из этих наборов вытягиваются общие primitives и foundations.
