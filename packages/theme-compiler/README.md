# packages/theme-compiler

Пакет для компиляции тем.

Его задача — брать токенный контракт и превращать его в форматы, которые можно использовать в приложениях и инструментах дизайна.

Будущие выходные форматы:

- CSS-переменные;
- фрагмент темы для Tailwind CSS 4;
- JSON-пресет для Playground;
- JSON, совместимый с Token Studio или похожим процессом;
- отчет миграции при изменении схемы токенов.

Этот пакет не должен содержать UI-компоненты. Он отвечает только за преобразование токенов.
