# packages/ui-react

React/Next.js-реализация дизайн-системы.

Пакет появится после инвентаризации и Figma-канонизации. До этого не переносим атомы преждевременно, чтобы не зафиксировать плохие API.

Будущая роль пакета:

- хранить стандартизированные React-компоненты;
- использовать Radix UI там, где он дает доступную структуру;
- сохранять shadcn/ui-совместимую семантику там, где это полезно;
- работать через семантические токены;
- быть пригодным для Next.js-проектов команды.
