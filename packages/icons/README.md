# packages/icons

Пакет для контракта иконок дизайн-системы.

Первый источник: Figma `Gigonom UI`, страница `Assets`, фрейм `Lucide`.

Позже пакет должен поддерживать:

- выбранные наборы иконок;
- правила замены иконок;
- маппинг между наборами;
- единые размеры;
- правила обводки и заливки;
- интеграцию с токенами.
