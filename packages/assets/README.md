# packages/assets

Пакет для политики и поставки ассетов: иллюстраций, изображений, декоративных материалов и motion-ассетов.

Пакет не должен становиться свалкой файлов.

У каждого ассета должны быть:

- источник;
- статус;
- назначение;
- правила использования;
- ограничения по лицензии, если они есть.
