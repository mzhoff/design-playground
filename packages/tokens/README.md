# packages/tokens

Источник правды для базовых и семантических дизайн-токенов.

Пакет должен хранить контракт токенов, а не разрозненные CSS-значения.

Здесь фиксируются:

- имена токенов;
- структура токенных слоев;
- типы токенов;
- допустимые значения там, где это возможно;
- базовые пресеты;
- правила совместимости.

Генерация CSS, Tailwind-мапы и Token Studio payload относится к `@design-playground/theme-compiler`.
